# PS1Screenshots
Power Shell Script to make screenshots every 10 secconds

# Usage

- Change the path where you want to save the pictures
- Change number of repetitions and sleep time in seconds
If you want to run the program for 2 hours and make a screenshot every 10 seconds, set the counter to 10 * 6 * 2 = 120
- Open CMD and run:
powershell.exe -windowstyle hidden -file ...\Screenshots\screenshot.ps1

Change the path, to the path of you program.
